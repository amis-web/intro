# 活動概要

　弊団体は、大和証券様の第32回ボランティア活動助成に採択していただいたことをご報告いたします。

　申請書には、現状取り組んでいる様々な特性や家庭事情を抱えた子どもへの支援の概要や、今後取り組んでいきたい事などへの熱意を書かせていただきました。

　日程が合わず、授賞式には参加できませんでしたが、今回の受賞を励みに今後ともボランティア活動に邁進してまいります。
