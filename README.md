# ボランティアサークル 公式Webサイト

## 概要
ボランティアサークルの公式ホームページです。
地域のイベント主催者および企業のCSR担当者に向けた、活動実績と信頼性をアピールするためのサイトです。

## ディレクトリ構成
- `index.html`: トップページ
- `assets/`: 画像、CSS、JavaScriptなどの静的ファイル
  - `css/`: スタイルシート
  - `js/`: スクリプト
  - `images/`: 画像素材

## 開発環境
- HTML5 / CSS3 / Vanilla JS
- 推奨エディタ: VS Code
- フォーマッター: Prettier

## ローカルでの確認方法
`index.html` をブラウザで開いてください。
（画像が表示されない場合は、Live Server等の拡張機能を使用することを推奨します）
